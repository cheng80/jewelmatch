# 스테이지 보상 체계

레벨 모드 클리어 시 `StageRewardEvaluator`가 `MatchBoardGameStats`, 목표 대비 점수 비율, 최대 콤보, 힌트 보존 여부를 함께 평가한다. 점수 단독 보상은 쓰지 않는다.

## 지급 규칙

| 보상 | 조건 |
|------|------|
| 룬 망치 | `removedGems >= 80` 또는 `matchGroups >= 14` 또는 목표 대비 `105%+`와 `removedGems >= 60` |
| 고대 폭탄 | `maxCombo >= 4` 또는 `removedSpecialGems >= 3` 또는 목표 대비 `110%+`와 `maxCombo >= 3` |
| 프리즘 변환 | `specialGemsCreated >= 4` 또는 하이퍼 생성 1회 이상 또는 목표 대비 `115%+`와 특수 보석 생성 2회 이상 |
| 토르 망치 | 특수 보석 발동 2회 이상 또는 별 보석 발동 1회 이상 또는 목표 대비 `112%+`와 특수 보석 발동 1회 이상 |
| 타임 슬립 | 유효 스왑 1~12회 그리고 `removedGems >= 45` |
| 운명의 셔플 | `matchGroups >= 18` 그리고 (`maxCombo >= 3` 또는 특수 보석 생성 2회 이상) |
| 힌트+ | 제한 모드에서 힌트 미사용 클리어 |
| 하이퍼 큐브 | `removedGems >= 120`와 `maxCombo >= 5`, 또는 목표 대비 `125%+`와 `maxCombo >= 4`와 특수 보석 생성 5회 이상 |
| 최소 보상 | 위 조건을 하나도 만족하지 못한 클리어는 토르 망치 1개 |

## 공통 정책

- 실패한 스테이지는 보상을 지급하지 않는다.
- 한 스테이지 보상 개수 제한은 없다.
- 같은 아이템을 여러 조건으로 받으면 수량을 합산한다.
- 같은 결과 화면에서 중복 지급되지 않도록 claim key로 1회만 지급한다.
