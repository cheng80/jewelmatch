# Progression Score Targets

진행 모드의 레벨별 목표 점수는 별도 데이터 테이블이 아니라
`lib/game/jewel_rank_progression.dart`의 `JewelRankProgression` 계산식에서 나온다.

## Formula

현재 코드의 핵심 상수는 다음과 같다.

```dart
static const int xpStep = 250000;
static const int scoreToXpScale = 100;
static const int scoreTargetScale = 3;
```

레벨별 목표 점수 계산식:

```text
xpNeededForNextLevel(level) = level * xpStep
scoreTargetForLevel(level) = (xpNeededForNextLevel(level) / scoreToXpScale) * scoreTargetScale
```

현재 상수 값을 대입하면:

```text
scoreTargetForLevel(level) = (level * 250,000 / 100) * 3
                           = level * 7,500
```

따라서 각 레벨의 목표 점수는 이전 누적 목표가 아니라, 해당 레벨에서 채워야 하는 단계 목표 점수다.
